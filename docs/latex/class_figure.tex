\doxysubsection{Detailed Description}
A class for representing and manipulating 2D geometric figures. 

The \doxylink{class_figure}{Figure} class provides functionality for creating, rendering, and manipulating simple 2D shapes within an Open\+GL context. It supports custom vertex definitions and interactive movement through the \doxylink{class_input}{Input} system.
