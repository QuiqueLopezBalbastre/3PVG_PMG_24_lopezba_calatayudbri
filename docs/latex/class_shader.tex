\doxysubsection{Detailed Description}
Represents an individual Open\+GL shader. 

The \doxylink{class_shader}{Shader} class encapsulates an Open\+GL shader object, which is a program that runs on the GPU. It provides functionality for loading, compiling, and managing different types of shaders (vertex, fragment, geometry).
