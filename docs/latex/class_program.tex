\doxysubsection{Detailed Description}
Represents an Open\+GL shader program. 

The \doxylink{class_program}{Program} class encapsulates an Open\+GL shader program, which is a collection of compiled shaders linked together to create a complete rendering pipeline. It provides methods for attaching shaders, linking the program, and setting uniform variables.
