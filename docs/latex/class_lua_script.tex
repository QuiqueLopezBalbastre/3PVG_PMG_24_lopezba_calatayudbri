\doxysubsection{Detailed Description}
Wrapper class for Lua scripting functionality. 

The \doxylink{class_lua_script}{Lua\+Script} class provides a high-\/level interface for loading and executing Lua scripts. It manages a Lua state and provides methods for running Lua code and handling errors.
