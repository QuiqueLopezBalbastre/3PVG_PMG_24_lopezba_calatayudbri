\doxysubsection{Detailed Description}
A multi-\/threaded job system for task parallelization. 

The \doxylink{class_job_system}{Job\+System} allows for the queuing and execution of different types of tasks with varying priorities across multiple worker threads. It\textquotesingle{}s designed to efficiently distribute workloads in areas like rendering, physics, AI, and audio processing.
