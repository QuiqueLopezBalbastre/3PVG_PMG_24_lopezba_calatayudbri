Detailed Description

Manages and abstracts user input through GLFW.

The Input class serves as a layer between the application and GLFW's input handling, providing a consistent interface for keyboard, mouse, and gamepad input. It also implements an action mapping system that allows binding high-level game actions to specific input keys.
