Detailed Description

A wrapper class for a GLFW window with OpenGL context.

This class encapsulates a GLFWwindow pointer and provides methods for window management, rendering, and input handling. It follows RAII principles to ensure proper cleanup of GLFW window resources upon destruction.

The class supports move semantics but not copy semantics to prevent multiple instances from managing the same window resources.
